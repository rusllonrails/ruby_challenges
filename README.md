Personal Ruby Challenges
==================================

[![CircleCI](https://circleci.com/gh/rusllonrails/ruby_metaprogramming_sandbox.svg?style=svg)](https://circleci.com/gh/rusllonrails/ruby_metaprogramming_sandbox)

## Dependencies

Ruby (2.6.4)

## Local Setup

```
gem install bundler:2.0.2
bundle
```

## Run Unit tests

```
bundle exec rspec .
```

## Run Rubocop check

```
bundle exec rubocop
```
